# Suicide_Discourse_Inidan_Institutions
# Behavioral Dynamics in Response to Extreme Events: A Reddit-Based Analysis

## Introduction
This project, for the course CGS616: Human Centered Computing, investigates whether there exists a **systematic trend in how human behavior evolves over time following an extreme event**. Specifically, we use **Reddit discussions as observable behavioral signals**, treating online discourse as a proxy for collective public response.

By analyzing how **language, sentiment, and interaction patterns shift over time**, the study aims to uncover the underlying evolution of public behavior in the aftermath of disruptive and emotionally significant events.

## Problem Statement
Extreme events often lead to rapid and structured shifts in public discourse. Understanding these behavioral transitions is important for identifying patterns in emotional processing, information dissemination, and collective action.

The central research question of this project is:

**Do people exhibit a systematic and time-dependent change in behavioral response following extreme events, as reflected through online discussions?**

To answer this, we model behavioral dynamics using Reddit data and track how discourse evolves across time windows following the event.

## Background & Motivation
In a previous study, we applied a similar framework to **natural disasters**, where we observed that such events trigger **structured and evolving discourse patterns**.

Individuals tended to transition across multiple response modes, including:

- **Information sharing** – exchange of facts, updates, and resources
- **Emotional expression** – fear, grief, anger, empathy, and solidarity
- **Action-oriented communication** – calls for help, donations, protests, or preventive measures

Building on these findings, the present project extends the framework to a more sensitive and socially significant context:

**suicides in educational institutions**

This allows us to study whether similar temporal behavioral structures emerge in discussions around institutional crises and mental health-related extreme events.

## Dataset
The dataset consists of **Reddit posts and comments scraped from discussions related to suicides in educational institutions**.

The collected data includes:

- Post titles
- Post body text
- Comments and replies
- Timestamps
- User interaction metadata
- Thread structure

These components help capture both **content evolution** and **interaction dynamics** over time.

## Objectives
The project aims to:

1. Identify temporal shifts in discourse following the event
2. Analyze sentiment and emotional progression
3. Detect recurring behavioral response states
4. Compare findings with prior disaster-event models
5. Understand collective online response patterns in educational crises

## Methodology
Our framework includes the following steps:

### 1. Data Collection
- Scraping relevant Reddit threads
- Filtering institution-related suicide discussions
- Cleaning and preprocessing textual data

### 2. Temporal Segmentation
- Dividing discussions into time windows
- Tracking progression from immediate to delayed response

### 3. Textual Analysis
- Sentiment analysis
- Topic modeling
- Keyword and phrase evolution
- Linguistic pattern analysis

### 4. Behavioral State Modeling
We classify discourse into response modes such as:

- informational
- emotional
- reflective
- action-driven

and study transitions between these states over time.

## Research Significance
This work contributes to understanding how communities process traumatic institutional events online.

The findings may help in:

- early detection of distress patterns
- institutional crisis monitoring
- policy intervention design
- mental health discourse analysis
